JSAPI for Mitao
============

for mitao h5 and app.

## install 

node 中使用

```
  npm i mitao-h5-api
```

mini文件直接使用

```
  git clone https://github.com/xuwu125/miTaoApi.git
  cd miTaoApi
  npm run mini
  cp lib/mitaoApi.mini.js 目标目录
```

## example 

目录 example 是测试目录，用来测试跟 h5之间的交互使用的，可以使用 vscode 的 live server 启动运行查看
